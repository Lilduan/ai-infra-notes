# AI Infra Learning Repo

This repository is a structured study workspace for transitioning from OS engineering to AI infrastructure.

## Layout

```
├── docs/               # 所有文档
│   ├── notes/          # 技术学习笔记（每天主题）
│   ├── daily/          # 每日日志（过程记录）
│   ├── weekly/         # 周总结与面试复盘
│   ├── templates/      # 笔记模板
│   ├── env.md          # 环境配置与复现步骤
│   └── memo.md         # 学习备忘录
├── code/               # 所有代码
│   ├── notebooks/      # Jupyter notebooks
│   ├── scripts/        # Shell 脚本
│   └── verify_torch_env.py
├── benchmarks/
│   ├── raw/            # 原始 benchmark 输出
│   └── reports/        # 整理后的分析报告
├── projects/           # 动手项目
└── references/         # 阅读笔记与文献摘要
```

## Working Rules

- Every study day should leave behind a note or benchmark artifact.
- Prefer measured results over vague impressions.
- Keep raw data and polished conclusions separate.
- At the end of each week, compress what you learned into material you can use in interviews.
